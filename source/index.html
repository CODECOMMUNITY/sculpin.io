PHP Static Site Generator

Sculpin is a static site generator written in PHP. It converts Markdown files and formats Twig templates into a set of static HTML files that can be easily deployed.

Why Sculpin?

Sculpin is a PHP static site generator. If PHP is your language of choice and you want to hack on a static site generator, you should feel right at home with Sculpin.

Sculpin runs Embedded Composer. If you love using Composer to manage dependencies, you'll love that Sculpin can manage your site's dependencies by way of Embedded Composer.

Sculpin uses the Twig template engine. If you are used to Twig and want to leverage its template inheritance, Sculpin was designed specifically with this in mind.

Sculpin is built around Symfony's HTTP Kernel. If you already know how to write a Symfony bundle, you already know how to extend Sculpin.
